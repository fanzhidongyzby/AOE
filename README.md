AOE
==========

项目名称
-----------

AOE网络演示

开发环境
-----------

Visual Studio 2010

详细说明
-----------

1. 设计实现了AOE网络的数据结构和关键路径的计算。
2. 操作步骤：
	* 按下E键，鼠标点击添加事件。
	* 按下A键，鼠标点击事件拖拽到另一合理事件添加活动。
	* 右键双击事件或者活动删除该事件或者活动。
	* 拖拽事件或者活动，悬停显示基本信息。
	* 空格键锁定和解锁屏幕拖动，滚轮键进行缩放。
	* 每次修改后重新进行拓扑排序，计算相应工期，并求出关键路径。
	* F11切换全屏，Esc推出全屏，鼠标中键按下还原平移缩放状态。
	* F1显示当前对话框。
